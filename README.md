# 💼 Salarios Ciberseguridad LATAM 2026

> Análisis visual e interactivo de empleos y salarios en ciberseguridad para LATAM — Mercado local vs Remoto US/EU

🔗 **[Ver demo en vivo → GitHub Pages](https://0xvanguard.github.io/latam-cybersecurity-salaries-2026/)**

## 📊 Contenido

- 6 roles analizados: SOC Analyst, Pentester, Cloud Security Engineer, Threat Hunter, Security Architect, CISO
- Rangos salariales en USD/año (Local LATAM vs Remoto US/EU)
- Gráficas interactivas con Chart.js
- Tabla de requisitos, certificaciones y rutas de carrera
- Diseño dark responsive

## 💰 Datos Clave

| Rol | Local LATAM | Remoto US/EU | Multiplicador |
|-----|------------|-------------|---------------|
| SOC Analyst | $12K–$25K | $30K–$55K | ~2.2x |
| Pentester | $18K–$35K | $45K–$80K | ~2.3x |
| Cloud Security Eng. | $25K–$45K | $60K–$100K | ~2.4x |
| Threat Hunter | $20K–$40K | $50K–$90K | ~2.5x |
| Security Architect | $35K–$55K | $70K–$120K | ~2.2x |
| CISO | $45K–$90K | $100K–$200K+ | ~2.5x |

## ⚠️ Fuentes

Estimaciones basadas en ACSMI, ISC2, Glassdoor 2025–2026. Varían según país, seniority y certificaciones.
