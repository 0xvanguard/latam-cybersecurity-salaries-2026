## 📌 ¿Para qué sirve este proyecto?

Si eres profesional de ciberseguridad en LATAM y quieres entender **cuánto puedes ganar**, **qué tan lejos estás del mercado remoto internacional** y **qué ruta seguir para llegar ahí**, este dashboard es para ti.

La brecha salarial entre trabajar localmente en Colombia, México o Argentina y trabajar remotamente para empresas de Estados Unidos o Europa puede ser de **2x a 4x**. Este proyecto lo visualiza con datos reales y actualizados.

---

## 📊 ¿Qué incluye el dashboard?

- **6 roles analizados:** SOC Analyst, Pentester, Cloud Security Engineer, Threat Hunter, Security Architect, CISO
- Rangos salariales en **USD/año** — Local LATAM vs Remoto US/EU
- Gráficas interactivas con **Chart.js**
- Tabla de **certificaciones y requisitos** por rol
- Rutas de carrera y **plataformas para conseguir trabajo remoto**
- Diseño dark responsive, sin dependencias externas

---

## 💰 Datos Clave: Local vs Remoto

| Rol | Local LATAM (USD/año) | Remoto US/EU (USD/año) | Multiplicador |
|-----|----------------------|------------------------|---------------|
| SOC Analyst | $12K – $25K | $30K – $55K | ~2.2x |
| Pentester / Red Team | $18K – $35K | $45K – $80K | ~2.3x |
| Cloud Security Engineer | $25K – $45K | $60K – $100K | ~2.4x |
| Threat Hunter | $20K – $40K | $50K – $90K | ~2.5x |
| Security Architect | $35K – $55K | $70K – $120K | ~2.2x |
| CISO | $45K – $90K | $100K – $200K+ | ~2.5x |

> Los datos representan rangos estimados para perfiles de nivel medio (3–7 años de experiencia). Varían por país, seniority y certificaciones.

## 🚀 ¿Cómo conseguir trabajo remoto en ciberseguridad?

### Plataformas recomendadas

| Plataforma | Tipo | Enfoque |
|------------|------|---------|
| [Toptal](https://www.toptal.com) | Freelance elite | Pentesting, AppSec, consultores |
| [Upwork](https://www.upwork.com) | Freelance | SOC, pentesting, auditorías |
| [Deel](https://www.deel.com) | Empleo remoto | Contratación full-time internacional |
| [Contra](https://contra.com) | Freelance | Independientes en tech/seguridad |
| [LinkedIn Jobs](https://linkedin.com/jobs) | Empleo remoto | Busca "remote" + "security" |
| [Wellfound (AngelList)](https://wellfound.com) | Startups | Roles de seguridad en startups US |

### Certificaciones que piden las empresas US/EU

- **Entrada:** CompTIA Security+, eJPT, Google Cybersecurity Certificate
- **Intermedio:** CEH, OSCP, CompTIA CySA+
- **Avanzado:** CISSP, CISM, GPEN, CRTE

### Perfil para el mercado internacional

- CV y perfil de LinkedIn **en inglés**
- Portafolio público en GitHub (write-ups, herramientas, proyectos)
- Participación en plataformas como **HackTheBox**, **TryHackMe**, **Bug Bounty (HackerOne/Bugcrowd)**
- Capacidad de comunicar hallazgos técnicos en reportes claros
